# Changelog

This changelog tracks releases of `@kitsuyui/react-use`.

`@kitsuyui/react-use` is a maintained fork of `react-use`. The inherited
upstream release history is preserved separately in
[UPSTREAM_CHANGELOG.md](./UPSTREAM_CHANGELOG.md) so links in that archived
history continue to point at the upstream repository that produced them.

## 17.6.0

- Started the maintained `@kitsuyui/react-use` package line from the upstream
  `react-use` 17.6.0 codebase.
- Published package metadata under `@kitsuyui/react-use` with this repository as
  the package homepage, repository, and issue tracker.

Future entries in this file should describe changes released from
`@kitsuyui/react-use`.
